Web Development Blog

Week 1

This week, I learnt about how to create a Github repository, how to clone a Github repository using Visual Studio Code, as well as how to commit the changes and push them to the remote Github repository and how to deploy a website using Github Pages.

Week 2

This week, I learnt about how to use HTML to define the layout of a webpage, as well as how to use CSS to style various elements on the webpage by changing style attributes such as their color and their border color for instance.

Week 3

This week, I learnt about flexboxes as well as media queries in CSS which could be used to enhance the responsiveness of a webpage by controlling how it would respond to certain changes such as changes in screen width and screen height. I also learnt about forms in HTML and how they could be used in a webpage.

Week 4

I learnt about Git and how it could be used for version control.

Week 5

This week, I learnt about the basics of Javascript, as well as about the browser console and linters.

Week 6

This week, I learnt about function based Objects and literal based Objects in Javascript, the differences between the "var" and "let" keywords and about for...of loops in Javascript. I also learnt about the DOM (Document Object Model) Tree.

Week 7

This week, I learnt about what events in Javascript were, as well as the different types of events such as UI events, as well as event listeners. I also learnt about localStorage in the browser.

Week 8

This week, I learnt about what JQuery was, as well as how to use it. I also learnt about what AJAX meant, and what APIs were.

Week 9

This week, I learnt about Bootstrap and what it could be used for as well as how it could assist in the website development process.

Week 12

This week, I learnt about what SQL and NoSQL databases are, as well as the differences between them. I also learnt about what RestDB is and how to set up a database in RestDB.

Week 13

This week, I learnt about best practices in web development, such as proper formatting of README Markdown files, as well as how to write proper commit messages, maintaining an organized file structure and good naming convention. I also learnt about how to ensure that my code is of good quality and not too incomprehensible or complicated for others to understand.

Week 14

This week, I learnt about what Lottie is and how to work with LottieFiles using Javascript code, as well as how to use the Lottie Editor.
